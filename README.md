
# 🚀 Habit Tracker App

A modern Habit Tracker web app built with **React**, **TypeScript**, and **Tailwind CSS**.

Track daily habits, edit tasks, search habits, and persist data using localStorage.

---

## ✨ Features

* ➕ Add habits
* 🗑️ Delete habits
* ✏️ Edit habit names
* ✅ Toggle completed habits
* 🔍 Search & filter habits
* 💾 localStorage persistence
* 🎯 Auto-focus editing input using `useRef`
* 📱 Responsive modern UI

---
